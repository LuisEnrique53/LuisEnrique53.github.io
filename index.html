Utilizando Python, creamos un minijuego muy simple. El juego se llama "GUESS THE NUMBER!!, este consiste en que tenemos un tablero digital el cual esta dividido por varias casillas. El jugador debe introducir un par de coordenadas para seleccionar la casilla que desea jugar. Una vez seleccionada la casilla, el jugador tendra 5 intentos para adivinar el número que esta dentro. Si responde incorrectamente el programa le dara una pista y le dara otro turno. Pero si acierta en el número, gana los puntos que se encuentran dentro de la casilla.

Utilizando el programa MATLAB, contruí varios proyectos. Sin embargo, voy a hablar especificamente de la aplicación del frenado magnetico, este programa es una simulación de un objeto que cae y es el frenado mediante inducción magnetica. Dentro del programa podemos observar que la simulación toma en cuenta diversos factores, como por ejemplo: el peso del objeto, material, velocida de caida, espesor, radio, etc; todo esto nos ayuda a generar un modelo más preciso y variable. Cabe mencionar, que para el desarrrollo de este programa se necesito un gran conocimiento de física para poder dar vida a esta simulación.
